# CodeAlpha_Calculator

A responsive calculator built with HTML, CSS, and JavaScript. The project supports all basic arithmetic operations, real-time display updates, and keyboard input, with a compact dark-themed interface.

## Features

- All four arithmetic operations — addition, subtraction, multiplication, and division
- Real-time display that updates with every keystroke, with the pending operation shown above the result
- Chained calculations (e.g. entering multiple operations in sequence)
- Clear (AC), backspace, and percent functions
- Error handling for invalid operations such as division by zero
- Keyboard support for digits, operators, decimal point, Enter/equals, Backspace, and Escape
- Button press animations and a responsive layout for smaller screens

## Tech Stack

- HTML5
- CSS3 (Grid, custom properties, animations)
- Vanilla JavaScript
